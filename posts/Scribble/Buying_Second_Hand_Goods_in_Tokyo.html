                                                                    东京邦歌烤谷捡垃圾不完全指北（2024 GW ver.）

                                                                    （这里邦指 BanGDream，歌指 Revue Starlight，烤指 Project Sekai）

                                                                    # 秋叶原

                                                                    # アキバのエックス

                                                                    • 在らしんばん对面，能捡邦的趴趴等。

                                                                    # らしんばん

                                                                    • 烤有两面墙，但是没有 25 时
                                                                    • 邦有一面墙
                                                                    • 歌只有几格，就在邦旁边和D5J 一起

                                                                    # ハビコロ（らしんばん楼上 3F 的谷店）

                                                                    • （这里其实是有两个中国姐妹想在らしんばん找 25 时找不到，被店员提醒可以去 3 楼找，所以我才去的）
                                                                    • 其中一家店有邦，无歌烤，比らしんばん东西多，但是店员态度疑似不是很友善啊
                                                                    • 另外有一家店有烤，个人感觉挺全面的，就是价格有点高， ena 立牌价格是animate 全新的同款的 1.7 倍。不过东西是真的多。

                                                                    # Animate

                                                                    • （都是新品，不属于捡垃圾的范畴，但是也写一下吧）
                                                                    • 按导向分楼层，一般向的 3F 有好几面墙的烤，男性向的 5F 有歌，有邦，连在一起。总之个人感觉比池袋的全。

                                                                    # 骏河屋

                                                                    • 秋叶原的骏河屋有非常多家，这里去了四家，其中一家是大楼，有非常多层，但是三者皆无。其中两家连在一起，都是模型。駅前店也是都没有。

                                                                    # kbooks

                                                                    • 品类非常非常齐全，歌都有一整面墙，烤邦甚至都各有五六面墙，甚至连 GBC 都有
                                                                    • 但是价格大部分都是正常价格或者溢价，小部分垃圾价，而且不能免税，但还是值得一看

                                                                    # 新宿

                                                                    # 骏河屋

                                                                    • 新宿的骏河屋超级超级大，邦和烤的都有，而且有专门的捡垃圾区，最高不超过300 円，在这家捡了非常多的垃圾！

                                                                    # 池袋

                                                                    # らしんばん

                                                                    • 在 animate 旁边有两家，有邦烤的是星巴克后面那一家，也是离 animate 更远的那一家

                                                                    # Store Mixa

                                                                    • （同样也不属于捡垃圾，但是值得一写，发现这家完全是出于意外，路过时看到广告牌有邦和 D5J 就知道应该进去了）
                                                                    • 邦歌非常非常多，而且是 Rosenchor 系列官方指定有特典的店，无需多言。不仅可以找到历年的场贩，还有最新最热的场贩（4.29 对邦和今年萝巡）

                                                                    # BOOKOFF

                                                                    • 邦有两行，但是并不是连在一起的，是分开的两行。
                                                                    • 烤有一面墙，而且这面墙的侧面也有，记得检查。
                                                                    • 记得检查附近的箩筐，可以捡一些大一点的东西（比如毛绒玩具和亚克力板毛巾等）

                                                                    # kbooks

                                                                    • GAME 馆有十几个货架的烤，一架 MyGO!!!!!
                                                                    • 在池袋有很多馆，但是都没有找到其他的，欢迎补充。

                                                                    # 中野

                                                                    # らしんばん

                                                                    • 邦有一面墙，东西挺齐全，有大量秋服立牌，在下面的箩筐还翻到了 980 円的 ppp 蝶对邦棒
                                                                    • 烤有一面墙，东西也挺齐全

                                                                    # まんだらけ （Mandrake）

                                                                    • （虽然不属于邦歌烤，但是属于捡垃圾）
                                                                    • 中野百老汇 3F 其中一家外面的墙全是 110 円一册 220 円三册的书，我直接怒买十九本，太便宜了
